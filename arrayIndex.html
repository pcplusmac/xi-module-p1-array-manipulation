how to create the first-class-function?

a function used as a parameter on it's name for other function -- a function being passed around note: alternatively, it can be assign to a const variable this variable is the name of the function, which means that on the right side of equator can be just an anonymous function contain the logic of the function. But without assigning function to a variable, it can be also used as first-class function by using it's name.

why do we do use this?

the reason for creating first-class-function is:

  • it is the function that can be pass around as the parameter for other function; such as 'loopThrough'
  • besides it can be passed around as a parameter, it can still keep the original feature as a function -- to be invoked by using it's name with parenthesis following by.

how to use it? the benefits of usage on it.

  • usage 1: inovke the method -- this is the normal usage of function name
  • usage 2: inject the function's logic to other function by passing this parameter to other function.
  • usage 3: increasing the flexibility for other function, this is alike the 'interface' in other languages, such as: handleArray() can take difference first-class methods to change the way of handling the array.
